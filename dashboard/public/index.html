The €2.3M Incident

A currency conversion bug caused €2.3M in losses when the system rounded BEFORE conversion instead of AFTER for zero-decimal currencies (CLP, COP).

❌ WRONG: €49.99 → €49.00 (rounded) → CLP 51,500
✅ CORRECT: €49.99 → CLP 52,614.48 → CLP 52,614 (rounded)
💸 Loss: ~1,114 CLP (~€1.04) per transaction
